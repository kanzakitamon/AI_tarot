# SE音源ファイルの配置

このフォルダに以下の3つの音源ファイルを配置してください：

## 必要な音源ファイル

1. **shuffle.mp3** - カードを混ぜる瞬間のSE
   - 内容：低音の「サラ…」「シャ…」という紙同士が触れる音
   - 音量：かなり小さめ（存在を意識しないレベル）
   - 目的：「処理開始」を耳で認識させる

2. **card-select.mp3** - カード選択時（タップ確定）のSE
   - 内容：極短い「コト」「トン」という木製・紙系の音
   - 高音禁止、余韻なし
   - 目的：選択の"重み"を出す

3. **card-flip.mp3** - カードがめくり切った瞬間のSE
   - 内容：かすかな「フワッ」「パサ…」＋微弱な低音
   - 回転開始では鳴らさない（180度で止まった瞬間のみ）
   - 目的：「めくった感」を脳に刷り込む

## 音源の方向性

- ✅ おすすめ：実写系効果音（紙・布・木）
- ✅ リバーブなし or 極薄
- ❌ 避ける：デジタルSE、魔法・ファンタジー系

## 音量設定

- iPhone音量50%時に「注意して聞くと分かる」レベル
- 目安：UI効果音の30〜40%
- アプリ内で自動的に音量調整されます（35%に設定済み）

## 注意事項

- 音源ファイルが存在しない場合、アプリは無音モードで動作します（エラーになりません）
- マナーモード時は自動的に無音になります
- 同時再生は禁止されています（前のSEが終わるまで待機）
